# 🚇 地铁涂鸦巡游 - Subway Graffiti Tour

一款基于 **Vue 3 + Vite + PixiJS** 的 H5 反应小游戏。在地铁线路上巡游，挑战喷涂判定和巡逻躲避，解锁皮肤，创造最高分！

---

## ✨ 游戏特色

### 🎮 核心玩法
- **线路地图系统** - 2 条地铁线路，16 个站点，自由探索
- **喷涂判定模块** - 缩圈式节奏判定，Perfect/Good/Miss 三档评分
- **巡逻躲避模块** - 警察视野锥 + 激光陷阱，考验反应与走位
- **连击系统** - 连续命中获得分数加成，挑战最高连击！

### 📊 数据统计
- 实时分数 / 最高分 / 累计总分
- Perfect / Good / Miss / 被抓次数
- 命中率统计、最大连击记录
- LocalStorage 持久化存档

### 👕 皮肤解锁
6 种涂鸦皮肤，通过累计分数解锁：
| 皮肤 | 解锁条件 |
|------|----------|
| 街头蓝 | 默认 |
| 烈焰红 | 2,000 分 |
| 霓虹绿 | 5,000 分 |
| 皇家紫 | 10,000 分 |
| 黄金色 | 20,000 分 |
| 宇宙粉 | 50,000 分 |

### 🔊 音频系统
- Web Audio API 程序合成音效（完美/良好/失误/被抓/连击等
- 动态背景循环音乐
- 可切换开关

## 🎯 游戏操作指南

### 主菜单
- **🚇 开始巡游** - 开始一轮新游戏
- **👕 皮肤** - 查看/选择已解锁的涂鸦皮肤
- **📊 统计** - 查看详细游戏数据

### 地图界面
- 点击已解锁的站点（彩色圆圈）前往挑战
- 🔒 灰色站点表示尚未解锁（通过完成前序站点或累计分数解锁）
- 小火车会沿线路动画移动到目标站点

### 站点挑战（每个站点包含 2 个阶段）

#### 阶段 1 - 喷涂挑战 (30秒)
```
目标：在缩圈与绿圈重合时点击获得判定
```
- 🎯 **Perfect (完美)** - 缩圈与绿色完美圈重合时点击 → +100分
- 👍 **Good (良好)** - 缩圈接近完美圈时点击 → +50分
- ❌ **Miss (失误)** - 缩圈超出范围或自动消失 → -20分，连击归零

#### 阶段 2 - 巡逻躲避 (20秒)
```
目标：在时间内不被警察发现，不触碰激光
```
- 👆 **移动** - 点击/滑动屏幕控制角色移动
- 💚 **安全区** - 角落绿色圆圈内绝对安全
- 👮 **警察视野** - 红色扇形区域，黄色闪烁时范围加倍！
- 🔴 **激光** - 先出现红色虚线警告，再变成实心红色激光（触碰即失败）

### HUD 信息
- 左上角：当前得分
- 中间上方：当前站点名称和阶段进度
- 右上角：历史最高分
- 右侧中间：连击数（2连击以上显示）
- 右下角：🔊/🔇 音频开关

## 💡 设计说明

### 模块解耦设计

| 模块 | 职责 |
|------|------|
| **线路地图** | 站点选择、小火车移动动画、解锁判定 |
| **喷涂判定** | 目标生成、缩圈动画、命中判定、粒子特效 |
| **巡逻躲避** | 巡逻兵 AI、视野锥、激光、安全区、碰撞检测 |
| **分数统计** | 分数计算、连击倍率、存档读写 |
| **皮肤解锁** | 皮肤配置、解锁判定、选择应用 |
| **音频模块** | 音效播放、背景音乐循环、音量控制 |

### 调参指南

在 `src/game/config.js` 中可以轻松调整游戏参数：

```javascript
// 喷涂判定参数
graffiti: {
  perfectRadius: 20,      // 完美判定圈半径
  shrinkSpeed: 120,     // 缩圈速度 (像素/秒)
  spawnInterval: 1200,   // 目标生成间隔 (毫秒)
  perfectScore: 100       // Perfect 基础分
}

// 巡逻躲避参数
patrol: {
  guardSpeed: 150,       // 警察移动速度
  flashRadius: 120,      // 视野范围
  caughtPenalty: 500       // 被抓扣分
}
```
